# Semantic HTML — Cheat Sheet

| Element        | What it is                                                           |
| -------------- | -------------------------------------------------------------------- |
| `<header>`     | Top of the page or section — logo, title, navigation                 |
| `<nav>`        | Navigation links — menus, breadcrumbs                                |
| `<main>`       | The primary content of the page — only ONE per page                  |
| `<section>`    | A thematic group of content, usually with a heading                  |
| `<article>`    | Self-contained content that could stand alone — blog post, news item |
| `<aside>`      | Indirectly related content — sidebar, related links                  |
| `<figure>`     | An image or illustration with an optional caption                    |
| `<figcaption>` | The caption for a `<figure>` — the small text below an image         |
| `<footer>`     | Bottom of the page — copyright, contact info, links                  |
